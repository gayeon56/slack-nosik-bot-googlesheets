# 김노식 ㅣ 노타 컬쳐의 수호자
이 슬랙앱은 2025년 5월 12일 만들어졌다. 
10개월 간 인턴으로 일하며 보고 느낀 노타 문화적 요소를 섞어내어 
전사 구성원에게 전달할 메시지를 작성했고, 주중 같은 시간 랜덤으로 메시지를 전송되도록 구성하였다.
